Contém um sistema de arquivos em C, utilizando noções de estrutura de dados e conceitos de sistemas operacionais, principalmente do linux, permite comandos básicos como:
* Criar/remover arquivos e diretórios;
* Visualizar direitórios e arquivos;
* Visualizar informações do sistema artifical.
Tem 80 blocos de memória, 128 bytes cada, cada arquivo tem direito a um bloco direto, que caso ultrapassado poderá usar blocos indiretos (Sem limite definido).
